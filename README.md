# 📸 Cozy Photobooth

A premium, aesthetic, browser-based photobooth experience built with **Next.js**. Capture, style, and download professional-grade photo strips directly from your browser.

> [!TIP]
> **Live Demo:** [cozy-photobooth.vercel.app](https://cozy-photobooth.vercel.app/)

---

## ✨ Overview

**Cozy Photobooth** is a modern reimagining of the classic physical photobooth, designed for the web. Inspired by nostalgic polaroid aesthetics, it provides an immersive and minimal interface for creating digital keepsakes.

Whether you're looking for a vintage film look or a playful sticker-filled collage, Cozy Photobooth offers a seamless, high-performance workflow that feels both familiar and state-of-the-art.

---

## 🚀 Features

- **🎯 Intelligent Workflow:** Guided experience from layout selection to final capture.
- **🖼️ Iconic Layouts:** Choose from classic Strips, Polaroid, Collage, or realistic Postcard formats.
- **🎨 Studio Suite:**
  - **Premium Filters:** Real-time filter engine (Warm Vintage, Soft Café, Golden Hour, etc.) with precision intensity control.
  - **Dynamic Decorations:** High-quality stickers and emojis with intuitive drag-and-drop placement.
  - **Artist Mode:** Integrated doodle layer for freehand drawing and personalization.
- **📸 High-Fidelity Capture:** Optimized for sharpness using 1080p source feeds and pixel-perfect canvas rendering.
- **📱 Fully Responsive:** Beautifully adaptive design that works perfectly on desktop, tablet, and mobile.
- **⚡ Instant Gratification:** One-click high-resolution PNG downloads with zero sign-up required.

## 🎯 Project Purpose

This project was developed as a deep-dive into:
*   **Browser Media APIs:** Implementing low-latency, high-resolution camera streams.
*   **Canvas Engineering:** Building a custom rendering engine for real-time filters and multi-layer compositions.
*   **UX/UI Craftsmanship:** Creating a "Cozy" design language that prioritizes visual excellence and smooth interactions.

---

## 📸 How It Works

1.  **Permission:** Grant secure camera access in your browser.
2.  **Selection:** Choose your desired layout and frame theme.
3.  **Capture:** Strike a pose! The booth handles the timing and filters.
4.  **Edit:** Head to the **Studio** to add stickers, doodles, and custom text.
5.  **Save:** Download your high-quality digital strip instantly.
